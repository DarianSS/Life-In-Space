The Challenge


The class will be divided in teams of 5 people. Your task is to research and develop a micro-website that describes a certain topic, by giving facts and trivia, and by using the previously taught lessons about web design. For each task you will receive further guidance that should help.


Constraints: Each task has a time limit of 50 minutes. Marking will be based on creativity, code formatting, and ingenuity in implementation. In other words, you have to find ways to make information attractive.

Research


For this project, you have to design a website that describes how life in space is, or might be for future inhabitants. You have to cover topics such as: difficulties for astronauts, communication, technologies used, food, leisure, and space suits.


For this research, probably most of you will use Google. Try to diversify your resources, by using trustworthy websites like The Verge or Mashable, and aim for scholarly articles when performing your search.

Intro to Making a Website



To design the layout of a website, half of the code is written in HTML5 and the other is written in CSS. This is because the HTML5 part is used to include the content, and the CSS part is used to stylise it and make it look nicer.


A website is made up of different parts such as a header that goes at the top and usually contains the title of the website, a content section where the posts go, a navigation bar which has links to other pages on the website and a footer which usually has a copyright statement.


Key things to know:


When writing the code for the website, it needs to have the following around the whole code:


<DOCTYPE html> 
<html> 
Code
</html>

The HTML5 part needs to have the following around the code:


<body> 
Code
</body>

The CSS part needs to have the following around the code:

<style> 
Code
</style>

The <div> element is used to create parts of the layout


Design the Header



The HTML5 code:


<div id="header"> 
    <h1>Title</h1>
</div>

The bold part can be changed to whatever you want as your title.


The CSS code:

#header {
    background-color:colour;
    color:colour;
    text-align:allignement;
    padding:sizepx;
    height:sizepx;
    border:sizepx solid #CCC;
    width:sizepx;
    margin:0px auto;
}

For colour, you can either write the name of the colour or look up the code of the exact colour you want from here: http://html-color-codes.info/


The margin:0px auto; part of the code matches the width of the header to the width of the content of the website.


.header-cont {
    width:100%;
    position:fixed;
    top:0px;
}

Adding this to the code makes the header fixed at the top of the webpage even when you scroll down.


Design the Navigation Bar



Here is basic html code for navigation bar:


<html>
    <head>
        <title>navigationbar</title>
        <style>
        </style>
    </head>
    <body>
        <div class="nav">
            <ul>
                <li><a href="#">homepage</a></li>
                <li><a href="#">page1</a></li>
                <li><a href="#">page2</a></li>
                <li><a href="#">page3</a></li>
                <li><a href="#">page4</a></li>
                <li><a href="#">page5</a></li>
                <li><a href="#">page6</a></li>
            </ul>
        </div>
    </body>
</html>

Now we can change the style of navigation bar by using CSS: 



.nav ul{
    width:980px;                    /*set width as 980px*/ 
    border:1px solid #000;          /*build a frame with color#000，width 1px*/
}

Then we can place the navigation bar in the middle 


.nav ul{
    width:980px;                /*set width980px*/
    border:1px solid #000;      /*set frame color#000，width 1px*/
    margin:0px auto 0px auto;   /*place in the middle also can be written as margin:0 auto*/
}

Also we can make the navigation bar horizontal:


.nav ul li{
    float:left;
}

Design your own navigation bar: 


.nav ul li a{
    width:80px;             /*set width80px*/
    height:28px;            /*set height28px*/
    line-height:28px;       /*set line height 28px*/
    background:red;         /*set background colour*/
    color:#FFF;             /*set text colour*/ 
    margin:5px 10px;
    font-size:12px;         /*front size*/
    display:block;
    text-align:center;      /*place the text middle*/
    text-decoration:none;
}

Actually, we can add some interaction on navigation bar, for example, when you put your mouse on the navigation bar it will change colour to highlight which you have selected. Here is the example code: 


.nav ul li a:hover{
    width:78px;
    height:26px;
    line-height:28px;
    border:1px solid red;
    color:red;
    background:#FFF;
}

Finally, you will need to link the buttons on your navigation bar to other pages. Example code: 


<a href=http://www.google.com>go to google</a>

Alternatively, you may want to link to a file. Example code: 


<a href=file:///f/baiduyundownload/an.jpg>picture</a>

The content



The content represents the body of your website. Inside it goes all your information and media, whether they are images, texts, tables, and so on.


Your task is to build this section in the most creative way possible. Include backgrounds, images, and watch out for ideas in the websites you use for your research. This task has allocated 100 minutes, so you have two classes to finish this part.


For this, you should use the HTML properties that you have learned in the previous lessons. Furthermore, you may embed a video to make things more easy to understand. If you want to embed something from popular websites, such as YouTube or Vimeo, you just have to click the “embed" button and copy that code in the place of your choosing (preferably in a <div>).



<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
</video>

As you already noticed, you have to include these properties:


  • width
  • height
  • controls (enables user input)

Design the Footer



Like in any other a document a footer is placed in the bottom of the page containing a document or a specific section of your website, to provide certain information. Typically this includes:

  • authorship and copyright information
  • contact information
  • sitemap
  • back to top links
  • related documents

Let’s take a look on how you can create on yourself!


To create a footer we use what is called the <footer> tag, which defines the footer for a section or a document. The following is a simple example on how to use it: 


<footer>
    <p>Posted by: John Doe</p>
    <p>Contact information: <a href="mailto:name@example.com"> name@example.com</a>.</p>
</footer>

Now try to style your footer using CSS like in the header example!


Tip: Here’s how to keep your footer at the bottom of your page:


HTML:

<body>
    <div id="body">
        <div id="header"> </div>
        <div id="content"> </div>
        <div id="footer"> </div>
    </div>
</body>

CSS:

html,
body {
    margin:0;
    padding:0;
    height:100%;
}
#body{
    min-height:100%;
    position:relative;
}
#header {
    padding:10px;
    background:#5ee;
}
#content {
    padding:10px;
    padding-bottom:80px;   /* Height of the footer element */
}
#footer {
    width:100%;
    height:80px;
    position:absolute;
    bottom:0;
    left:0;
    background:#ee5;
}

Now you’re ready to create your own footer!
